# Contributing

Soul Spec is maintained by [ClawSouls](https://github.com/clawsouls). We welcome contributions!

## Ways to Contribute

### Report Issues
Found a bug or have a suggestion? [Open an issue](https://github.com/clawsouls/soulspec/issues).

### Improve Documentation
This docs site is in the [soulspec repo](https://github.com/clawsouls/soulspec). PRs welcome â€” click "Edit this page" on any doc.

### Propose Spec Changes
For changes to the spec itself:
1. Open a discussion in [GitHub Discussions](https://github.com/clawsouls/clawsouls/discussions)
2. Describe the use case and proposed change
3. If accepted, submit a PR to the spec markdown files

### Create Community Souls
Build and publish souls to [clawsouls.ai](https://clawsouls.ai). The best way to contribute is to grow the ecosystem.

## Development Setup

```bash
git clone https://github.com/clawsouls/soulspec.git
cd soulspec
npm install
npm start
```

## Code of Conduct

Be respectful, constructive, and inclusive. We follow the [Contributor Covenant](https://www.contributor-covenant.org/).
